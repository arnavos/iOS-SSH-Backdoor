# iOS SSH Backdoor

⚠️ For educational purposes only.

This project contains a shell script that demonstrates how an SSH backdoor could be configured on an iPhone.  
It is intended strictly for **educational purposes only** and for learning about device security, system access, and SSH configuration.

The script automates several steps that would normally be done manually when configuring SSH access.

---

## Features

For educational purposes only, the script performs the following actions:

- Creates a new user on the device
- Creates a directory inside the user's home folder
- Copies an SSH binary into that directory
- Changes the binary permissions so it can run
- Creates a launch daemon so the binary runs automatically
- Generates an SSH key
- Disables password authentication
- Restarts the SSH service

This repository is provided **for educational purposes only** to demonstrate how such mechanisms work and how they might be detected or secured against.

---

## Usage

For educational purposes only.

1. Connect your iPhone to your computer using a USB cable.
2. Open a terminal.
3. Navigate to the folder containing the script.
4. Run the script: ./ssh-backdoor.sh


This demonstration is provided **for educational purposes only** to help understand SSH configuration and mobile device security.
